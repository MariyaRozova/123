Программа расшифровывает сообщения закодированные с помощью шифра Цезаря и выполняет, при надобности, обратную операцию. 
Реализует возможность расшифровки сообщения при наличии или отсутствии информации о шаге, при котором применялось кодирование. Далее закодированное сообщение подлежит «расшифровке» грубой силы из 26 возможных вариантов.
